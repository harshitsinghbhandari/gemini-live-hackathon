The Problem

Autonomy without accountability
is just chaos with good PR.

AI agents are powerful. That's exactly the problem. Every delegated task is a potential irreversible action taken without your explicit consent.

🔓

Agents act first

Without a gate, an agent interpreting "clean up my inbox" might delete three years of email before you can blink.

🌀

Context gets lost

Agents operate in isolation. They don't know what's irreversible, what's sensitive, or what requires human judgment.

👁️

No audit trail

When something goes wrong, there's no record of what the agent did, when, and why. Accountability disappears.

🔑

No real gate

Confirm dialogs can be bypassed. A trust layer must be structural — biometric, not just a button.

The Security Model

Three tiers. One clear line.

Every action Aegis takes is classified before it executes. Not as an afterthought — as a first principle.

Green · Silent

Read-only

Runs without interruption. Nothing written, nothing changed.

AUTH REQUIRED: None

Read emails and threads
List calendar events
View documents and sheets
Browse GitHub issues and PRs
🟢screenshots/tiers/green.png
Yellow · Confirm

Creates & modifies

Aegis reads the action aloud. You say yes or no.

AUTH REQUIRED: Voice Confirmation

Create and update calendar events
Create and edit documents
Reply to email threads
Open GitHub issues, add labels
🟡screenshots/tiers/yellow.png
Red · Biometric

Irreversible

Touch ID on Mac. Face ID on iPhone. No bypass.

AUTH REQUIRED: Touch ID / Face ID

Send, delete, or trash emails
Merge or delete repositories
Delete calendar events
Clear tasks, delete sheets

Principles

Not a wrapper.
A philosophy.

🔐

Classify before execute

Every action is risk-classified before it runs. The gate is structural, not optional.

🎙️

Voice-native

Built on Gemini Live API. Talk to your agent naturally — confirmation flows are part of the conversation.

🧬

Biometric-gated

Your fingerprint and your face are the final keys. No bypass path exists for irreversible actions.

📡

Real-time audit log

Every action streams to your dashboard via SSE. The record is permanent, the visibility is instant.

📱

Cross-device control

Your Mac delegates, your iPhone approves. The security boundary spans your devices intentionally.

⚡

Five-minute setup

One install command, a few API keys, and you're running a fully secured agentic environment.
